# My 101 Dreams

A personal dream and life-goal tracking app inspired by the idea of creating a list of 101 dreams and gradually turning them into reality.

Built with vanilla HTML, CSS and JavaScript.

## Live Demo

[Open the app](https://YOUR_USERNAME.github.io/YOUR_REPOSITORY_NAME/)

---

## Features

### Dreams
- Create, edit and delete dreams
- Pin important dreams
- Mark dreams as completed
- Track progress through subgoals
- Add completion dates
- Automatic progress calculation
- Priority system (A / B / C)

### Categories
- Default categories
- Create custom categories
- Custom category colors
- Edit and delete categories

### Filtering & Search
- Search dreams by name
- Filter by:
  - status
  - category
  - priority
- Quick stat-card filtering

### UX & UI
- Responsive mobile-first design
- Sticky navigation/search area
- Dark mode support
- Clean card-based layout
- Animated progress bars
- Confirmation dialogs for destructive actions

### Data
- LocalStorage persistence
- Export backup as JSON
- Import backup from JSON
